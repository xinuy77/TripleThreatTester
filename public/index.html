Triple Threat

This web application is for testing authentication system, Triple Threat.

Triple Threat is the authentication system designed with three concatenated words, taken from a pre-existing list of 128 words, allowing for repetition and in which orders matter. This allows for 221 potential passwords in total, which contributes to both the scheme's overall effectiveness and safety.
